Collect Lead Form plugin. Simple but flexible.

== Description ==

Collect Lead Form is designed to collect lead easily, or use as a contact form. The form supports Ajax-powered submitting. 
If you want to modify or add new form or add CAPTCHA, please contact the owner faferdaus@gmail.com.

= Collect Lead Form needs your support =

It is hard to continue development and support for this free plugin without contributions from users like you. If you enjoy using Collect Lead Form and find it useful, please consider to donate to faferdaus@gmail.com. Your donation will help encourage and support the plugin's continued development and better user support.

= Privacy notices =

With the default configuration, this plugin, in itself, does not:

* track users by stealth;
* send any data to external servers;
* use cookies.

== Installation ==

1. Upload the entire `collect-lead-form` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the **Plugins** screen (**Plugins > Installed Plugins**).

You will find **Collected Leads** menu in your WordPress admin screen.
